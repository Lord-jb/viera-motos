# Projeto Website - Avelloz Motos (Viera Moto Center Marabá)

## 1. Descrição

Este é um projeto de front-end para o website institucional da **Avelloz Motos - Viera Moto Center Marabá**. O objetivo é criar uma vitrine digital premium, moderna e totalmente responsiva para apresentar a linha de motocicletas 2025 da marca (AZ1, AZ125 ALFA, AZ160 XTREME).

O design foi inspirado na estética limpa e sofisticada de marcas premium (como Triumph Motorcycles), adaptado à identidade visual da Avelloz (cores: Azul `#0D1B2A` e Laranja `#FF6F00`).

---

## 2. Stack de Tecnologia

Este projeto foi desenvolvido utilizando **tecnologias web puras**, sem o uso de frameworks ou bibliotecas externas (exceto Google Fonts), para garantir máxima performance e leveza.

* **HTML5:** Para a estrutura semântica do conteúdo.
* **CSS3:** Para estilização, layout (Flexbox e Grid) e responsividade (Media Queries).
* **JavaScript (ES6+):** Para interatividade do usuário, como o menu hambúrguer e o seletor de cores dos modelos.
